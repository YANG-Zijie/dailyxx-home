# 隐私政策 / Privacy Policy

版本 / Version: 2025-08-04

<p align="center">
  <a href="#zh">中文</a> | <a href="#en">English</a>
</p>

## 中文

{: #zh}

「今日打卡」尊重并保护用户隐私：

1. **不收集、不上传** 任何个人数据。
2. 所有习惯与打卡记录仅存储于用户本地设备。
3. 当用户主动保存庆祝截图时，应用仅请求“照片库写入”权限；除写入图像外，不会访问照片库其他内容。

## English

{: #en}

DailyXX respects and protects user privacy:

1. **No collection or upload** of any personal data.
2. All habits and check-in records are stored only on the user's device.
3. When the user saves a celebration screenshot, the app only requests Photo Library write permission; it does not access any other photo library content.
